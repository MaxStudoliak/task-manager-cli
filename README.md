# Task Manager CLI

Прості CLI-інструмент для керування задачами (додавання, редагування, видалення, пошук, експорт).

## 🚀 Запуск

```bash
npm install
npm start
```

Або просто:

```bash
node index.js
```

## 🧭 Меню CLI

При запуску програма покаже меню з опціями:

1) Додати задачу
2) Показати всі задачі
3) Редагувати задачу
4) Видалити задачу
5) Знайти задачу
6) Експортувати задачі
0) Вийти

> Задачі зберігаються у файлі `data/tasks.json`.

## 📝 Приклади використання

### 1) Додавання задачі

- Введіть назву
- (необов'язково) Опис
- (необов'язково) Дедлайн (`YYYY-MM-DD`)
- (необов'язково) Пріоритет (`low`, `medium`, `high`)
- (необов'язково) Статус (`true` / `false`)

### 2) Пошук задачі

Використовується пошук за ключовим словом у назві та/або описі задачі.

### 3) Експорт

Після вибору експорту програма попросить вибрати формат:
- `json` → створює `tasks-export.json`
- `csv` → створює `tasks-export.csv`

## 🧩 API (core)

Функції у папці `src/`:

- `addTask(taskData)` — додає задачу
- `getTasks(filters, sortBy)` — повертає список задач (`filters` може містити `priority`, `status`, `search`)
- `updateTask(index, updates)` — змінює задачу за індексом (0‑based)
- `deleteTask(index)` — видаляє задачу за індексом
- `clearTasks()` — очищає список задач

### Валідація

Функції у `src/utils.js`:

- `checkTitle(title)`
- `checkDeadline(deadline)`
- `checkPriority(priority)`
- `checkStatus(status)`
